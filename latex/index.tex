Header-\/only.}{Thread-safe container for sharing data between threads. Header-only.}}\label{index_autotoc_md1}

\begin{DoxyItemize}
\item Thread-\/safe push and fetch.
\item Use stream operators to push (\texorpdfstring{$<$}{<}\texorpdfstring{$<$}{<}) and fetch (\texorpdfstring{$>$}{>}\texorpdfstring{$>$}{>}) items.
\item Value type must be default constructible.
\item Blocking (forever waiting to fetch).
\item Range-\/based for loop supported.
\item Close to prevent pushing and stop waiting to fetch.
\item Integrates well with STL algorithms in some cases. Eg\+: std\+::move(ch.\+begin(), ch.\+end(), ...).
\item Tested with GCC, Clang, and MSVC.
\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Requirements}{Requirements}}\label{index_autotoc_md2}

\begin{DoxyItemize}
