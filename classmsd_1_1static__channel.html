Detailed Description

template<typename T, std::size_t Capacity>
class msd::static_channel< T, Capacity >

Thread-safe container for sharing data between threads.

Allocates elements on the stack. Does not throw exceptions. Implements a blocking input iterator.

Template Parameters
TThe type of the elements.
CapacityThe maximum number of elements the channel can hold before blocking.
